# 🏏 IPL Analysis 2008–2025 | Power BI Dashboard

## 🎯 Project Overview
**IPL Analysis 2008–2025** is an **interactive Power BI dashboard** designed to analyze and visualize the performance of IPL teams and players from **2008 to 2025**.  
The project focuses on delivering **dynamic insights**, allowing users to explore **team and player statistics** season by season with ease.

---

## 🧩 Business Objective
The main goal of this project is to build a **comprehensive and interactive Power BI dashboard** that:

- Provides insights into **team and player performances** across multiple IPL seasons.  
- Enables users to **select any IPL season** using slicers/filters for instant visualization and analysis.  

---

## 📊 Primary KPIs

### 🏆 Winner & Runner-Up Insights
- **Winner Team** of the selected season — dynamically displays **team name and logo**.  
- **Runner-Up Team** of the selected season — dynamically displays **team name and logo**.  

---

## 📈 Season Stats (Dynamic KPIs)
For any selected IPL season, the dashboard dynamically displays detailed player statistics:

### 🟧 Orange Cap Stats
- Orange Cap holder name  
- Total runs scored  
- Team name the player represented  
- Player image (dynamically rendered)

### 🟪 Purple Cap Stats
- Purple Cap holder name  
- Total wickets taken  
- Team name the player represented  
- Player image (dynamically rendered)

### 🟩 Total Fours in Season
- Player name  
- Total fours hit  
- Team name the player represented  
- Player image (dynamically rendered)

### 🟦 Total Sixes in Season
- Player name  
- Total sixes hit  
- Team name the player represented  
- Player image (dynamically rendered)

## 📂 Dataset Overview
The dataset contains IPL match-level and player-level data from **2008 to 2025**, including:
- Team names and match outcomes  
- Player performance stats (runs, wickets, fours, sixes, etc.)  
- Season/year information  
- Award details (Orange Cap, Purple Cap, etc.)  

> *(Data used is for educational and analytical purposes only.)*
